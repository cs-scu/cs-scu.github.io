کارگاه آموزشی Git و GitHub

یک دوره جامع برای مدیریت ورژن و همکاری تیمی

انجمن علمی علوم کامپیوتر با افتخار برگزار می‌کند: کارگاه مقدماتی تا متوسطه Git و GitHub. این دوره برای تمام دانشجویانی طراحی شده است که می‌خواهند مهارت‌های خود را در زمینه مدیریت ورژن، که یکی از ضروری‌ترین ابزارها در دنیای نرم‌افزار است، تقویت کنند.

نقل قول از مدرس دوره: "یادگیری Git دیگر یک انتخاب نیست، بلکه یک ضرورت برای هر توسعه‌دهنده‌ای است."

سرفصل‌های دوره

در این کارگاه مباحث زیر به صورت کامل پوشش داده خواهد شد:

  • نصب و راه‌اندازی اولیه Git
  • مفاهیم اصلی مانند Commit, Branch, Merge
  • کار با GitHub، ایجاد Pull Request و بازبینی کد
  • حل تعارض‌ها (Conflict Resolution)
git commit -m "یک کامیت نمونه برای نمایش در خبر"
